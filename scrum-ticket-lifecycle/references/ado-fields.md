# ADO 欄位對應

| Scrum 概念 | ADO 欄位 |
|-----------|---------|
| AC（驗收條件） | Description 或 Acceptance Criteria 欄 |
| DoD 檢查清單 | Checklist（Task 內的子項目） |
| Sprint 歸屬 | Iteration Path |
| Squad 歸屬 | Area Path |
| 跨 Squad 主責標記 | Tags，格式：`owner-squad:S4` |
| 插單紀錄 | Discussion 欄（必須留文字記錄） |
| 允收紀錄 | Discussion 欄（記錄允收原因與 Owner） |
| Carry-over 標記 | Tags，格式：`carry-over` |
| 估點（人天） | Story Points 欄 |
| Block 標記 | Blocked（ADO 內建狀態或 Tags） |
